Textual Use Case: Book Flight

Primary Actor: Authenticated User

Precondition: User is logged in; Flights exist in DataStore.

Main Success Scenario:

  1. User selects "Search Flights" and enters "Lahore".
  2. System displays available Flights via FlightService.
  3. User selects Flight ID and clicks "Book".
  4. System validates capacity and calculates 5% fee.
  5. System appends Booking and Bill records to DataStore.

Alternate Flow: (Insufficient Capacity) System displays error and returns to search.
